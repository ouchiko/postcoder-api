postcode-api
Welcome to the postcode-api service.
Access to this service is simple. Make a request to the API endpoint with a postcode (or partial) and it will return details about the postcodes found. Partial postcodes must be a minimum of 4 characters and be a valid syntax (letters and numbers only)
Example request string
/postcode/:postcodestring:
Example response
{
  "data": [
    {
      "address": {
        "address1": "Harpenden",
        "address2": "St. Albans",
        "address3": "Hertfordshire",
        "area": "Eastern",
        "country": "England",
        "postcode": "AL5 3HG"
      },
      "lat": 51.8288573766,
      "lng": -0.380464939137
    }
  ],
  "search": "AL53HG"
}
Development
Mainly produced in Python running in a Docker container. Data from OS Open Data, converted using OS Open Name to CSV.
